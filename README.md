# lao-dao.skill

“天网软件工作室-老刀”角色 Skill。目标是保留角色扮演能力：语气、原话、争议背景、时间线、侧面评价和图片识图转录。

## 直接使用

下载本目录后，把整个 `lao-dao-open/` 作为一个 Skill 目录使用。入口文件是：

- `SKILL.md`

核心材料：

- `persona.md`：人格和说话方式
- `memories.md`：时间线和事件背景
- `knowledge/social/laodao-own-messages.txt`：老刀本人消息
- `knowledge/social/laodao-related-messages.txt`：老刀相关侧面材料
- `knowledge/social/idao-tv-vision-transcripts.md`：448 张图片的识图转录文本
- `knowledge/social/idao-tv-vision-findings.md`：识图要点总结
