# 原始文本参考

本目录保留的是“可开源文本语料”，用于增强扮演能力和追溯判断依据。原始图片和本机归档不随仓库发布，但已转成文本。

## 优先级

1. `social/laodao-own-messages.txt`
   - 老刀本人消息和截图补录原话。
   - 用于学习口癖、脏话密度、标点、辈分压制、对线节奏。

2. `social/idao-tv-vision-transcripts.md`
   - `i刀TV` 448 张图片的识图转录文本。
   - 用于补充老刀原话、价格/引流/鸡蛋/创建令牌等截图证据。
   - 该来源偏黑料收集，老刀本人原话权重大于他人评价。

3. `social/laodao-related-messages.txt`
   - 多人提到老刀的侧面材料。
   - 用于理解别人如何评价他，但不能把单一发言者视角当 100% 事实。

4. `social/laodao-summary.md`
   - 自动汇总的老刀消息档案。
   - 用于快速定位高频话题和代表性原话。

5. `social/idao-tv-vision-findings.md`
   - 从图片识图中抽出的要点。
   - 用于快速加载 ls 渠道归属、QQ 引流、鸡蛋/令牌等纠正点。

## 使用规则

- 做角色扮演时，先读 `persona.md`，再读 `memories.md`，最后按需要查这些原始文本参考。
- 需要复现语气时，优先查 `laodao-own-messages.txt` 和 `idao-tv-vision-transcripts.md`。
- 需要判断事件事实时，区分“本人原话”“用户纠正”“他人侧面评价”。
